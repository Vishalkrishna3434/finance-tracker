# 💰 Personal Finance Tracker

Full-stack MERN dashboard to track income, expenses, and portfolio performance with real-time analytics.

![Status](https://img.shields.io/badge/Status-In%20Development-yellow)
![Stack](https://img.shields.io/badge/Stack-MERN-blue)

Coming soon

## ✨ Features
- JWT Authentication (register/login/logout)
- Track income and expense transactions
- Real-time profit/loss calculations
- Interactive charts (income vs expenses, spending by category)
- REST API with protected routes
- Responsive dashboard UI

## 🛠 Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| React.js | Node.js | MongoDB Atlas |
| Chart.js | Express.js | Mongoose |
| React Router | JWT + Bcrypt | |

## ⚙️ Setup
```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd client
npm install
npm start
```
